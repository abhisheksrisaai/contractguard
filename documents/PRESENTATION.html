🧠 The AI Pipeline

1. RAG — Retrieval-Augmented Generation

Contract clause → Embedding vector → Search Qdrant → Retrieve similar fair clauses → LLM comparison

2. LLM Risk Analysis (Groq Llama 3.3 70B)

Identifies one-sided terms, hidden obligations, legal red flags, unfair penalties

3. Semantic Embeddings (TF-IDF + SVD)

384-dimensional vectors · Cosine similarity · Lightweight — fits 512MB RAM
